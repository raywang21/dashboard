# 动态表单组件使用说明

## 功能特性

这个表单组件实现了您要求的所有功能：

### 简单字段
- ✅ **文本输入框** - 单行文本输入
- ✅ **密码输入框** - 密码类型输入，自动隐藏内容
- ✅ **下拉单选** - 从预定义选项中选择一个
- ✅ **下拉多选** - 从预定义选项中选择多个，支持 Chip 标签显示
- ✅ **开关** - 布尔值切换
- ✅ **多行文本框** - 支持多行文本输入

### 复杂字段
- ✅ **数组(string)** - 动态添加/删除字符串数组项
- ✅ **数组(integer)** - 动态添加/删除数字数组项
- ✅ **对象-固定属性** - 固定结构的对象，支持嵌套
- ✅ **对象-允许任意键（动态键值对）** - 支持动态添加键值对

### 嵌套属性
- ✅ **对象层级关系** - 支持多层嵌套对象，使用 Accordion 折叠面板展示

### 布局功能
- ✅ **响应式布局** - 使用 CSS Grid 实现响应式布局
- ✅ **分组字段** - 使用 Card 组件对字段进行分组
- ✅ **字段换行** - 支持字段在不同屏幕尺寸下的换行显示

## 数据结构

表单数据会实时更新并显示在页面底部的"实时数据预览"区域。数据结构示例：

```json
{
  "username": "用户名",
  "password": "密码",
  "role": "user",
  "permissions": ["read", "write"],
  "enabled": true,
  "description": "描述文本",
  "tags": ["标签1", "标签2"],
  "numbers": [1, 2, 3],
  "profile": {
    "name": "姓名",
    "email": "邮箱",
    "phone": "电话"
  },
  "customProperties": [
    { "key": "自定义键1", "value": "自定义值1" },
    { "key": "自定义键2", "value": "自定义值2" }
  ],
  "address": {
    "street": "街道",
    "city": "城市",
    "country": {
      "name": "国家名称",
      "code": "国家代码"
    }
  }
}
```

## 使用方法

1. **填写表单** - 在相应的字段中输入数据
2. **添加数组项** - 点击"添加文本"或"添加数字"按钮
3. **删除数组项** - 点击数组项旁边的删除图标
4. **添加键值对** - 点击"添加键值对"按钮
5. **展开嵌套对象** - 点击 Accordion 标题展开/折叠
6. **提交表单** - 点击"提交表单"按钮，数据会打印到控制台
7. **重置表单** - 点击"重置表单"按钮清空所有数据

## 技术实现

- **React 19** - 使用最新的 React 版本
- **TypeScript** - 完整的类型支持
- **Material-UI 7** - 现代化的 UI 组件库
- **响应式设计** - 支持桌面和移动设备
- **状态管理** - 使用 React Hooks 管理表单状态

## 扩展性

组件设计具有良好的扩展性：

1. **添加新字段类型** - 在 `FieldConfig` 接口中添加新的类型
2. **自定义验证** - 可以添加字段验证逻辑
3. **自定义布局** - 支持更复杂的布局配置
4. **API 集成** - 可以轻松集成后端 API

## 注意事项

- 表单数据会实时更新并显示在预览区域
- 提交时会弹出提示，数据会打印到浏览器控制台
- 所有字段都支持默认值配置
- 嵌套对象使用 Accordion 组件展示，避免界面过于复杂
